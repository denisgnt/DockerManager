# 🎨 Описание интерфейса Docker Manager

## Общий вид приложения

### Верхняя панель (AppBar)
```
┌─────────────────────────────────────────────────────────────┐
│ Docker Manager      API: 10.174.18.242:2375          🔄     │
└─────────────────────────────────────────────────────────────┘
```
- **Заголовок:** "Docker Manager" слева
- **Информация об API:** адрес Docker API в центре
- **Кнопка обновления:** иконка Refresh справа

---

## Список контейнеров

Контейнеры отображаются в виде **адаптивной сетки карточек**:
- **Десктоп:** 3 карточки в ряд
- **Планшет:** 2 карточки в ряд  
- **Мобильный:** 1 карточка в ряд

### Карточка контейнера

```
┌─────────────────────────────────────────────────┐
│  nginx-web-server              ● running        │
│                                                  │
│  Image: nginx:latest                            │
│  ID: a1b2c3d4e5f6                              │
│  Ports: 80:8080, 443:8443                       │
│  Создан: 02.11.2025, 14:30:15                  │
│                                                  │
│  ⏹️  🔄  🗑️                           👁️       │
└─────────────────────────────────────────────────┘
```

**Элементы карточки:**

1. **Заголовок:**
   - Имя контейнера (слева)
   - Статус с цветом (справа)

2. **Информация:**
   - Образ Docker
   - Короткий ID контейнера (12 символов)
   - Проброшенные порты
   - Дата и время создания

3. **Кнопки управления:**
   - ▶️ **Play** (зеленая) - запустить контейнер
   - ⏹️ **Stop** (красная) - остановить контейнер
   - 🔄 **Restart** (оранжевая) - перезапустить
   - 🗑️ **Delete** (красная) - удалить контейнер
   - 👁️ **View** (синяя) - просмотр логов

**Цветовые индикаторы статуса:**
- 🟢 **Зеленый (success)** - running (запущен)
- 🔴 **Красный (error)** - exited (остановлен)
- 🟡 **Желтый (warning)** - paused (на паузе)
- ⚪ **Серый (default)** - другие статусы

## Модальное окно логов

Открывается при нажатии на кнопку 👁️ (View)

```
┌───────────────────────────────────────────────────────────┐
│  Логи контейнера: nginx-web-server      ● Подключено    ✕ │
├───────────────────────────────────────────────────────────┤
│                                                             │
│  ┌─────────────────────────────────────────────────────┐  │
│  │ 2025-11-02 14:30:15 Server started                  │  │
│  │ 2025-11-02 14:30:16 Listening on port 80            │  │
│  │ 2025-11-02 14:30:20 GET /index.html 200 OK         │  │
│  │ 2025-11-02 14:30:21 GET /style.css 200 OK          │  │
│  │ ...                                                  │  │
│  │ ...                                                  │  │
│  │ ...                                                  │  │
│  └─────────────────────────────────────────────────────┘  │
│                                                             │
├───────────────────────────────────────────────────────────┤
│                            [Очистить]      [Закрыть]      │
└───────────────────────────────────────────────────────────┘
```

**Элементы окна логов:**

1. **Заголовок:**
   - Название: "Логи контейнера: [имя]"
   - Индикатор подключения (зеленый/красный)
   - Кнопка закрытия ✕

2. **Область логов:**
   - Темный фон (#1e1e1e) - стиль терминала
   - Моноширинный шрифт (monospace)
   - Логи в реальном времени через WebSocket
   - Автоматическая прокрутка к последним записям
   - Поддержка многострочных логов

3. **Кнопки действий:**
   - **Очистить** (желтая) - очистить отображаемые логи
   - **Закрыть** (синяя) - закрыть окно

## Уведомления

### Уведомление об ошибке
```
┌─────────────────────────────────────────────────┐
│ ⚠️  Ошибка подключения к Docker API             │
└─────────────────────────────────────────────────┘
```
- Красный фон
- Иконка предупреждения
- Текст ошибки
- Кнопка закрытия

### Индикатор загрузки
```
        ⏳ Загрузка...
```
- Отображается при первой загрузке
- Крутящийся индикатор (CircularProgress)
- Центрирован на странице

### Пустое состояние
```
┌─────────────────────────────────────────────────┐
│                                                  │
│         Контейнеры не найдены                   │
│                                                  │
└─────────────────────────────────────────────────┘
```
- Отображается когда нет контейнеров
- Серый текст по центру

## Цветовая схема (Темная тема)

### Основные цвета
- **Фон приложения:** #121212 (темно-серый)
- **Фон карточек:** #1e1e1e (чуть светлее)
- **Текст основной:** #ffffff (белый)
- **Текст вторичный:** #b0b0b0 (серый)

### Акцентные цвета
- **Primary (синий):** #2196f3 - основные действия
- **Secondary (розовый):** #f50057 - акценты
- **Success (зеленый):** #4caf50 - успешные статусы
- **Error (красный):** #f44336 - ошибки и удаление
- **Warning (оранжевый):** #ff9800 - предупреждения

## Интерактивные элементы

### Hover эффекты
- **Карточки:** поднимаются на 4px при наведении
- **Кнопки:** изменяют яркость
- **Иконки:** меняют цвет

### Transitions
- **Карточки:** плавное движение (0.2s)
- **Модальное окно:** fade-in анимация
- **Списки:** плавное появление элементов

### Tooltips
При наведении на кнопки управления появляются подсказки:
- "Запустить"
- "Остановить"
- "Перезапустить"
- "Удалить"
- "Просмотр логов"

## Особенности UX

✅ **Автообновление** - список контейнеров обновляется каждые 5 секунд  
✅ **Подтверждение удаления** - перед удалением показывается confirm dialog  
✅ **Автоскролл** - логи автоматически прокручиваются к последним записям  
✅ **Индикатор подключения** - показывает статус WebSocket соединения  
✅ **Обработка ошибок** - все ошибки отображаются в виде Alert уведомлений  
✅ **Адаптивный дизайн** - работает на всех размерах экранов  
✅ **Темная тема** - приятная для глаз цветовая схема
