# Обновление v1.2.0 - Оптимизация под MUI 7 и переменные окружения

## 🎯 Основные изменения

### ✅ Обновление до Material UI 7
- Обновлены все компоненты для использования новых API MUI 7
- Заменено `InputProps` на `slotProps` в TextField
- Заменено `PaperProps` на `slotProps.paper` в Dialog
- Оптимизированы все `sx` пропсы для использования сокращений MUI 7
- Улучшена производительность рендеринга компонентов

### 🔧 Переменные окружения
Добавлена поддержка конфигурации через `.env` файл:

```env
VITE_DOCKER_API_HOST=10.174.18.242
VITE_DOCKER_API_PORT=2375
VITE_PORT=3001
VITE_BACKEND_PORT=5001
```

**Что изменилось:**
- ✅ Порт Vite dev-сервера настраивается через `VITE_PORT`
- ✅ Docker API адрес вычитывается из `VITE_DOCKER_API_HOST` и `VITE_DOCKER_API_PORT`
- ✅ Backend порт настраивается через `VITE_BACKEND_PORT`
- ✅ Отображение API адреса в интерфейсе использует переменные окружения
- ✅ Создан файл `.env.example` с шаблоном настроек
- ✅ Добавлен `.env` в .gitignore

### 🎨 Улучшения UI в плиточном представлении

#### Лучшее выравнивание карточек
- **Grid responsive:** xs=12, sm=6, md=4, lg=3 (лучшая адаптивность)
- **Заголовки:** исправлено переполнение длинных имен с `wordBreak: 'break-word'`
- **Chip статуса:** добавлено `flexShrink: 0` для предотвращения сжатия
- **CardContent:** улучшено расстояние между элементами с `gap: 1`
- **CardActions:** добавлен разделитель (`borderTop`) для визуального разделения
- **Иконки:** унифицирован размер всех иконок до `small`
- **Typography:** оптимизированы стили для лучшей читаемости

#### Дополнительные улучшения
- Добавлены анимации: `transition: 'transform 0.2s, box-shadow 0.2s'`
- Улучшен hover эффект для карточек
- Лучшее выравнивание элементов с использованием flexbox
- Оптимизированы отступы и паддинги

### 📱 Адаптивность

#### App.jsx
- API адрес скрывается на мобильных устройствах: `display: { xs: 'none', sm: 'block' }`
- Текст кнопок переключения режима скрывается на мобильных
- Улучшена компактность тулбара на маленьких экранах

### 🔍 Улучшения поиска
- Локализован placeholder: "Поиск по имени, образу или ID..." → русский
- Добавлена подсказка когда поиск не дал результатов
- Обновлено использование `slotProps.input` вместо `InputProps`

## 🚀 Миграция с v1.1.0

### Шаг 1: Создайте .env файл

```bash
cp .env.example .env
```

### Шаг 2: Настройте переменные окружения

Откройте `.env` и укажите свои значения:

```env
VITE_DOCKER_API_HOST=ваш_хост
VITE_DOCKER_API_PORT=ваш_порт
VITE_PORT=3001
VITE_BACKEND_PORT=5001
```

### Шаг 3: Перезапустите приложение

```bash
npm run dev
```
